# CS2 Skin Changer - Standalone Edition

> **Complete in-one-file skin modification tool - No installation required!**

## 🎮 Features

✅ **No Installation** - Just .exe and done!  
✅ **Auto-Launch CS2** - Automatically launches the game  
✅ **In-Game Overlay** - Press INSERT to open menu  
✅ **Real-Time Skins** - Change skins instantly  
✅ **Stealth Mode** - Anti-detection protection  
✅ **Auto-Attach** - Automatically connects to CS2  

## 🚀 Quick Start

### Download
1. Download `skinchanger.exe` from Releases
2. That's it! No installation needed

### Usage
1. **Double-click** `skinchanger.exe`
2. **Wait** for CS2 to launch (30 seconds max)
3. **Press INSERT** to open menu
4. **Select weapon and skin**
5. **Enjoy your skins!**
6. **Press ESC** to exit

## 📋 What it does

1. ✅ Checks if CS2 is running
2. ✅ If not, launches it from Steam automatically
3. ✅ Waits for game to fully load
4. ✅ Injects skin changer into game
5. ✅ Attaches overlay menu
6. ✅ Ready to change skins!

## 🎮 Controls

| Key | Action |
|-----|--------|
| **INSERT** | Open/Close overlay |

## 🆘 Troubleshooting

### "CS2 not found"
- Install Counter-Strike 2 from Steam
- Run the program again

### "Permission denied"
- Run as Administrator
- Disable antivirus temporarily (false positive)

### "Overlay not showing"
- Press INSERT again
- Try restarting the program
